# PLM-BOM 获取技能

## 功能概述

本技能提供使用 Chrome MCP 直连方式从 PLM 系统获取成品料号和完整 BOM 数据的能力。

### 核心功能

| 功能 | 单个操作 | 批量操作 | 保存格式 |
|------|---------|---------|---------|
| 根据产品型号查成品料号 | ✅ | ✅ | Excel |
| 根据成品料号获取完整 BOM | ✅ | ✅ | Excel + JSON |

---

## 使用方法

### 1. 根据产品型号查询成品料号

#### 单个查询
```
请查询产品型号 N3D 的成品料号
```

#### 批量查询
```
请查询以下产品型号的成品料号：
N3D, NX31, WF2785C
```

### 2. 根据成品料号获取完整 BOM

#### 单个查询
```
请获取成品料号 30007324 的完整 BOM
```

#### 批量查询
```
请获取以下成品料号的完整 BOM：
30007324, 30007392, 30007433
```

## 技术实现

### 系统 URL

```
PLM 物料查询：http://oa.netcoretec.com/wui/index.html#/main/cube/search?customid=78&menuIds=-92,-103&menuPathIds=-92,-128,-103&_key=2sv8b5
PLM BOM 查询：http://oa.netcoretec.com/wui/index.html#/main/cube/search?customid=80&cj=1&_key=qhvac1
```

### 操作流程

#### 查询成品料号

1. 打开 PLM 物料查询页面
2. 在"客标型号"输入框输入产品型号
3. 等待下拉列表加载，选择完全匹配的型号
4. 点击搜索按钮
5. 等待数据加载
6. 提取表格数据
7. 筛选成品料号（3000 开头）
8. 保存为 Excel

#### 获取完整 BOM

1. 打开 PLM BOM 查询页面
2. 在"顶层料号"输入框输入成品料号
3. 按回车搜索
4. 等待 BOM 数据加载
5. 提取完整 BOM 数据
6. 保存为 JSON 和 Excel

## 注意事项

### 1. 产品型号下拉选择

**问题**：搜索产品型号时可能出现多个相似选项（如 N3D, N3D-V2, N3D-Pro）

**解决方案**：
- 优先选择与输入完全匹配的选项
- 如果没有完全匹配，提示用户确认
- 不要选择近似型号

### 2. 等待时间

| 操作 | 建议等待时间 |
|------|------------|
| 页面初始加载 | 5 秒 |
| 输入操作后 | 2 秒 |
| 搜索/查询后 | 5 秒 |
| 复杂 BOM 加载 | 5-10 秒 |

### 3. 文件保存路径

```
C:\Users\Administrator\.openclaw\plm-exports\
├── {产品型号}_成品料号清单_{日期}.xlsx
└── BOM\
    ├── {料号}\
    │   ├── BOM_{料号}_data.json
    │   └── BOM_{料号}_{日期}.xlsx
    └── 批量获取汇总_{日期}.xlsx
```

## 脚本说明

### scripts/search_part_number.py

根据产品型号查询成品料号并保存为 Excel。

**主要函数**：
- `save_part_numbers_to_excel()`: 保存料号数据为 Excel
- `filter_finished_goods()`: 筛选成品料号（3000 开头）

### scripts/get_bom.py

根据成品料号获取完整 BOM 并保存为 Excel 和 JSON。

**主要函数**：
- `save_bom_to_excel_and_json()`: 保存 BOM 数据
- `extract_bom_from_html()`: 从 HTML 表格提取 BOM 数据

## 常见问题

### Q1: 搜索产品型号时没有下拉选项？
**A**: 检查输入是否正确，或尝试点击输入框后再输入。

### Q2: BOM 数据加载不完整？
**A**: 增加等待时间至 10 秒，或检查网络状态。

### Q3: 如何区分成品料号和半成品料号？
**A**: 
- 成品料号：3000 开头
- 半成品料号：2000 开头
- 原材料：1000 开头

### Q4: 批量获取时某个料号失败？
**A**: 继续处理其他料号，最后汇总时会显示失败记录。
