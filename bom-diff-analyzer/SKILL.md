# BOM 差异分析 Skill

基于产品描述解析和逐层 BOM 比较，自动识别多型号产品之间的 BOM 差异，并智能判断差异的合理性（正常/异常/需确认）。

## 功能描述

- **多型号 BOM 对比**：以标准料号为基准，比较其他料号的 BOM 差异
- **逐层分析**：按层级（1-5）逐一比较物料组成
- **智能判断**：基于产品描述（品牌、地区、芯片方案等）自动判断差异合理性
- **Excel 报告**：输出详细的差异对比表

## 使用场景

### 适用场景

- 多型号产品 BOM 对比分析
- BOM 一致性校验
- 物料标准化评估
- 客户定制差异分析

### 前置条件

- Python 3.10+
- BOM 文件（命名格式：`料号_BOM_*.xlsx`）
- 产品描述文件（Excel，含物料号和描述列）

## 输入输出

### 必填参数

| 参数 | 说明 |
|------|------|
| `--bom-directory` (-d) | BOM 文件目录 |
| `--products-file` (-p) | 成品料号清单文件 |
| `--standard-pn` (-s) | 标准料号（比较基准） |
| `--output-directory` (-o) | 输出文件目录 |

### 可选参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--output-filename` (-f) | `01_BOM 差异详细对比表.xlsx` | 输出文件名 |

### 命令示例

```bash
python skill.py \
    --bom-directory "C:\PLM\NX30\By_CPU\MT7981B" \
    --products-file "C:\PLM\NX30\NX30_products_20260331_151559.xlsx" \
    --standard-pn "30006913" \
    --output-directory "C:\PLM\NX30" \
    --output-filename "01_BOM 差异详细对比表.xlsx"
```

## 差异判断规则

### 异常

- 产品描述完全相同，但 BOM 有差异

### 正常

- 品牌不同 + 包装/PCBA/认证文件差异
- 地区不同 + 电源/认证要求差异
- 规格不同 + 结构件差异
- 芯片方案不同 + 层级 5 元器件差异
- 客户定制差异（层级 2-4）
- 组装版本差异（层级 1 + ASS）

### 需确认

- 品牌不同但地区相同 + 认证文件
- 品牌不同 + 保修贴/说明书（功能可能相同）
- 其他需人工审核的场景
