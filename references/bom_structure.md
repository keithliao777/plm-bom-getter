# BOM 数据结构说明

## BOM 层级结构

### 典型 BOM 层级

```
Level 1: 成品料号（3000 开头）
  ├─ Level 2: ASS 组件（2000 开头）
  │    ├─ Level 3: DIP 组件
  │    │    └─ Level 4: SMT 组件
  │    │         └─ Level 5: 电子元器件
  │    └─ Level 3: 结构件
  └─ Level 2: 包装材料
```

### 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 层级 | BOM 层级 | 1, 2, 3, 4, 5 |
| 父阶料号 | 父级物料编号 | 30007324 |
| 顺序号 | 顺序编号 | 0010, 0020 |
| 子阶料号 | 子级物料编号 | 10022645 |
| 物料描述 | 物料详细描述 | 外置电缆，网线，CAT5e... |
| 用量 | 使用数量 | 1, 0.10 |
| 优选等级 | 物料优选状态 | 优选，可选，慎选，禁选 |
| 位号 | PCB 位号 | U11, C40 |
| 替代组 | 替代料组 | A1, B |
| 组合 | 组合标识 | A, B |

## 物料分类

### 按料号分类

| 分类 | 料号前缀 | 说明 |
|------|---------|------|
| 成品 | 3000xxxx | 可直接销售的产品 |
| 半成品 | 2000xxxx | ASS、DIP、SMT 组件 |
| 原材料 | 1000xxxx | 电子元器件、结构件 |
| 包材 | 5000xxxx | 贴纸、条码等 |

### 电子元器件分类

- **IC**: 10018xxx, 10019xxx
- **电容**: 10000xxx
- **电阻**: 10001xxx
- **电感**: 10019xxx
- **连接器**: 10018xxx, 10021xxx

## 数据验证

### BOM 完整性检查

1. **层级连续性**: 确保层级从 1 到 N 连续
2. **父子关系**: 子阶料号的父阶必须存在
3. **用量合理性**: 用量应为正数

### 常见数据问题

| 问题 | 说明 | 解决方案 |
|------|------|---------|
| 层级跳跃 | 从 Level 1 直接到 Level 3 | 检查中间层级是否缺失 |
| 孤儿节点 | 父阶料号不存在 | 检查数据完整性 |
| 循环引用 | A 是 B 的子件，B 是 A 的子件 | 数据错误，需要修正 |
