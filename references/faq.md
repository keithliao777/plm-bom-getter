# 常见问题解决方案

## 问题分类

### 1. 连接问题

#### Q: 无法访问 PLM 系统
**原因**: 
- 不在内网环境
- 网络故障

**解决方案**:
1. 确认连接到公司内网
2. 检查网络连接
3. 联系 IT 部门确认系统状态

---

### 2. 搜索问题

#### Q: 搜索产品型号时没有结果
**可能原因**:
1. 产品型号输入错误
2. 该产品型号不存在
3. 下拉列表未正确选择

**解决方案**:
1. 检查产品型号拼写
2. 尝试搜索部分型号（如"N3D"而不是"N3D-V2"）
3. 确认下拉列表中选择了正确的型号

#### Q: 下拉列表有多个相似选项
**示例**: 输入"N3D"，显示 N3D, N3D-V2, N3D-Pro

**解决方案**:
1. 优先选择与输入**完全匹配**的选项
2. 如果没有完全匹配，向用户确认
3. 不要选择近似型号

---

### 3. BOM 获取问题

#### Q: BOM 数据加载不完整
**可能原因**:
1. 等待时间不足
2. 网络延迟
3. BOM 数据量过大

**解决方案**:
1. 增加等待时间至 10 秒
2. 检查网络连接
3. 分批次获取大型 BOM

#### Q: BOM 行数为 0
**可能原因**:
1. 料号输入错误
2. 该料号没有 BOM
3. 权限不足

**解决方案**:
1. 确认料号正确（应为 3000 开头）
2. 在 PLM 系统中手动验证
3. 联系管理员确认权限

---

### 4. 数据保存问题

#### Q: Excel 文件保存失败
**可能原因**:
1. 目录不存在
2. 文件被占用
3. 权限不足

**解决方案**:
1. 确认输出目录存在
2. 关闭已打开的文件
3. 检查文件写入权限

#### Q: JSON 文件乱码
**原因**: 编码问题

**解决方案**:
```python
# 确保使用 UTF-8 编码
with open(json_path, 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=2)
```

---

### 5. 批量处理问题

#### Q: 批量获取时某个料号失败
**解决方案**:
1. 继续处理其他料号
2. 记录失败的料号
3. 最后生成汇总报告时标注失败记录

#### Q: 批量处理速度过慢
**优化方案**:
1. 减少单个批次的数量（建议 5-10 个）
2. 适当减少等待时间
3. 使用并行处理（如果系统支持）

---

## 错误代码处理

### TimeoutError
```python
try:
    browser.act(...)
except TimeoutError:
    # 重新获取快照
    browser.snapshot(...)
    # 使用新 ref 重试
    browser.act(...)
```

### ElementNotFound
```python
try:
    browser.act(ref="e273", ...)
except Exception as e:
    # 页面可能已刷新，重新获取快照
    browser.snapshot(...)
    # 查找新 ref 并重试
```

## 最佳实践

### 1. 单个查询
- 适合快速查询单个料号
- 响应速度快
- 建议用于临时查询

### 2. 批量查询
- 适合获取多个料号
- 自动生成汇总报告
- 建议用于定期数据同步

### 3. 数据验证
- 获取后验证数据完整性
- 检查 BOM 行数是否合理
- 对比历史数据确认一致性
